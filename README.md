# VisualFlow

An Electron UI for plotting serial data.

## Installation

1. Clone this repository.

```bash
git clone git@github.com:lukasvasadi/visualflow.git
```

2. Navigate to the project directory and install the dependencies.

```bash
npm install
```

3. Run the start script.

```bash
npm start
```
